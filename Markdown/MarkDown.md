## 四、强调语法

粗体的用法：在两侧各使用两个星号隔开 **我是粗体**

斜体的用法：在两侧各使用一个星号隔开 *我是斜体*

要加粗并斜体显示：在两侧各使用三个星号隔开 ***同时使用粗体和斜体***

## 六、列表语法

### 有序列表

1. 创建有序列表，请在每个列表项前添加数字并紧跟一个英文句点
2. 数字不必按数学顺序排列
3. 但是列表项应当以数字1开始

### 无序列表

- 要创建无序列表
- 请在每个列表项前添加
- 破折号- 星号* 或加号+

### 在列表中嵌套其他元素

1. 在保留列表连续性的同时在列表中添加另一种元素
2. 请将该元素缩进**四个空格**或**一个制表符**
  
    > 我是缩进的元素

3. 我还是列表

## 七、代码语法

要将单词或短语表示为代码，请将其包裹在反引号中

`我是代码`

如果要表示的内容中本身就包含反引号，则用双反引号``包裹

``我是代码，`我本身就带有反引号`，所以用双反引号包裹``

### 围栏式代码块

三个反引号或者三个波浪号围起来

# MarkDown扩展语法

## 一、表格

### 要添加表，请使用三个或多个连字符（---）创建每列的标题，并使用管道（|）分隔每列。您可以选择在表的任一端添加管道。

| Syntax      | Description |
| ----------- | ----------- |
| Header      | Title       |
| Paragraph   | Text        |

Tip: 使用连字符和管道创建表可能很麻烦。为了加快该过程，请尝试使用[Markdown Tables Generator](https://www.tablesgenerator.com/markdown_tables)。

### 对齐

您可以通过在标题行中的连字符的左侧，右侧或两侧添加冒号（:），将列中的文本对齐到左侧，右侧或中心。

| Syntax      | Description | Test Text     |
| :---        |    :----:   |          ---: |
| Left        | Title       | Right         |
| Paragraph   | Text        | And more      |

## 二、脚注

### 要创建脚注参考，请在方括号[^1]内添加插入符号和标识符。标识符可以是数字或单词，但不能包含空格或制表符。
